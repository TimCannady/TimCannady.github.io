This is what happens when asked to choose just one topic from many! On one hand I think it's best to do a deeper dive in one than skim a a few. ActiveRecord

I began with knowing nothing about AR. I only chose it because I've seen the name a few times. And I remember having a feeling of, "this sounds important." So let's get at it:

ActiveRecord is what's called an ORM, or an Object-Relational Mapping. Its name refers to the Active Record Pattern identified by Martin Fowler. So it's kind of a program whose name shares liklihood with an industry theory, if that makes sense. It's also the "M" in MVC, being Model-Control-Viewer - something I know is important but am yet to get learn about.

What does AR do? Quite a lot. And it's somewhat hard for me to understand all the benefits because I'm new to SQL and have never touched Ruby - both of which AR works closely with. That said, I understand the parts where it does the translations between Ruby objects and the database that houses them. For example, let's say a website rents out DVDs and the owner wants to display the most recent. Assuming they're already listed in his database, he'd normally have to do a significant SQL query to sort by release date and return the most recent. This is possible because AR provides an object-oriented interface the database, making it much easier to interact with. And that's about it! RegEx

RegEx: that thing we're never allowed to use! Just kidding though. Well, sort of. There's something to be said for a tool that a lot of people say is best used in moderation.

'RegEx' is short for 'regular expression,' and is all about pattern-matching. It's typically used to search for something in a data set. For example, you might want to remove all the spaces in a string, remove the zeroes in an array, or convert exclamation points to double-exclamation points. So it's worth noting that RegEx can search for any character type and is traditionally used to either search for, or update, the result.

I think the reason it's suggested to use in moderation is because programs empower us with tools, like loops and iterators, that are potentially better to use. At least academically speaking.

RegEx statements typically use delimiters like '/', so when you see something similar to /[a-w&&[^c-g]z]/, think RegEx.
